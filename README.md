## ğŸš€ Getting Started

### Prerequisites

No special prerequisites needed! Just a modern web browser.

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Soumyadeepta04/Netlix-Clone.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd Netlix-Clone
   ```

3. **Open the project**
   - Simply open `index.html` in your web browser
   - Or use a local development server like Live Server (VS Code extension)

### Using Live Server (Recommended)

1. Install the Live Server extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"
4. The project will open in your default browser with live reload capability
